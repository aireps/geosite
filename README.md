# geosite

Custom geosite.dat for Xray/V2Ray routing.

Fork of [hydraponique/roscomvpn-geosite](https://github.com/hydraponique/roscomvpn-geosite) — all upstream categories preserved.

## Added categories

| Category | Description | Source |
|---|---|---|
| `category-ip-geo-detect` | ~144 IP geolocation domains | [v2fly/domain-list-community](https://github.com/v2fly/domain-list-community/blob/master/data/category-ip-geo-detect) |
| `ipip` | ipip.net, myip.la | dependency for the above |
